Chatbot Widget Test

Widget Status:

If you see a chat icon or widget in the bottom corner, the widget is working!

Check the browser console (F12) for any error messages.

Instructions:

  1. Open browser developer tools (F12)
  2. Go to the Console tab
  3. Look for any error messages
  4. Check if the widget appears in the bottom corner

Expected Behavior:

  • A chat widget should appear in the bottom right corner
  • You should be able to click on it to open a chat interface
  • No JavaScript errors should appear in the console
